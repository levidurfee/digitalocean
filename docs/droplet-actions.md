# Droplet Actions

** Available Methods**

* `enableBackups`
* `disableBackups`
* `reboot`
* `powerCycle`
* `shutdown`
* `powerOff`
* `powerOn`
* `restore`
* `passwordReset`
* `resize`
* `rebuild`
* `rename`
* `changeKernel`
* `enableIPv6`
* `enablePrivateNetworking`
* `snapshot`
* `retrieve`

You can chain some of the methods, but you must pass the request each time. This may end up changing, but the
API should remain the same.
